# Sales Data Analysis using SQL (SQLite)

## Project Objective
Analyze retail sales data to evaluate sales performance, profitability, trends, and product-level insights to support data-driven business decisions.

## Dataset
Retail Superstore sales dataset containing 51,290 sales records.

## Tools
- SQL (SQLite)
- DB Browser for SQLite

## Analysis Performed
- Sales and profitability KPIs
- Monthly sales and profit trends
- Regional performance analysis
- Category performance analysis
- Top-performing products
- Loss-making products
- Discount impact analysis

## Key Insights
- Sales show a steady upward trend with strong performance in Q4.
- The Central region generates the highest total revenue.
- Technology category leads in both sales and profitability.
- Several products generate negative profit despite high sales volume.
- Higher discount levels are associated with lower profit margins.
